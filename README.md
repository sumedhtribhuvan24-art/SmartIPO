SmartIPO 🚀
DRHP-Based IPO Intelligence Platform

SmartIPO is a fintech-focused IPO intelligence platform designed to simplify IPO research and DRHP analysis for retail investors and finance enthusiasts.

The platform provides live IPO tracking, DRHP-based insights, analytics, and an interactive interface to help users better understand upcoming IPOs without going through lengthy financial documents manually.

📌 Features
📈 Live IPO fetching and tracking
📄 DRHP-based company analysis
🤖 AI/Chatbot integration for investor assistance
📊 IPO analytics and financial insights
⚡ Fast and responsive frontend
🐳 Docker support for deployment
🔍 Simplified user-friendly interface for IPO research
